Game Description

osu! is a freeware rhythm game developed by Dean "peppy" Herbert. Game levels are called beatmaps. In each beatmap, a song will play and different Beats hit elements (hit circles, sliders and spinners) will appear on the play field. Players must use a pointing device to manipulate these elements in time with the playing song's rhythm in order to earn points. The use of a graphics tablet in combination with a keyboard is common in osu!, although some players do use a mouse.
